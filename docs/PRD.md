# PRD: 전자 근로계약서 서비스 (kimcontract)

## 1. 개요

### 1.1 프로젝트 목적
고용주와 근로자가 온라인으로 표준근로계약서를 작성하고 서명할 수 있는 서비스. 블록체인(Solana)을 활용하여 계약서의 위변조를 방지하고 무결성을 보장한다.

### 1.2 주요 타겟
- 소규모 사업장 (편의점, 카페, 음식점 등)
- 아르바이트 고용이 빈번한 업종
- 일용직/단기 근로자 고용 사업장

## 2. 핵심 기능

### 2.1 계약서 작성 (고용주)
- 표준근로계약서 양식 기반 폼 입력
- 계약 유형 선택: 정규직(REGULAR), 파트타임(PARTTIME), 일용직(DAILY)
- 근무 조건 설정
  - 근무 요일 (다중 선택)
  - 근무 시간 (시작/종료)
  - 휴게 시간
  - 시급
  - 급여 지급일

### 2.2 계약서 전송 및 서명
- 고용주 서명 → 근로자에게 전송 (이메일/링크)
- 근로자 계약 내용 확인 및 서명
- 양측 서명 완료 시 계약 체결

### 2.3 PDF 생성 및 다운로드
- 서명 완료된 계약서 PDF 자동 생성
- 고용주/근로자 모두 다운로드 가능

### 2.4 블록체인 검증
- PDF 해시값 생성 (SHA-256)
- Solana 블록체인에 해시 기록
- 트랜잭션 ID로 원본 검증 가능

## 3. 사용자 플로우

```
[고용주]                           [근로자]
   │                                  │
   ├─ 계약서 작성                      │
   ├─ 고용주 서명                      │
   ├─ 계약서 전송 ──────────────────→ │
   │                                  ├─ 계약 내용 확인
   │                                  ├─ 근로자 서명
   │ ←─────────────────── 서명 완료 ──┤
   │                                  │
   ├─ PDF 다운로드                    ├─ PDF 다운로드
   ├─ 블록체인 검증                   ├─ 블록체인 검증
```

## 4. 계약서 상태 (Status)

| 상태 | 설명 |
|------|------|
| DRAFT | 작성 중 (저장만 됨) |
| SENT | 고용주 서명 완료, 근로자에게 전송됨 |
| SIGNED | 근로자 서명 완료 |
| COMPLETED | 블록체인 기록 완료, 계약 체결 |

## 5. 기술 스택

### 5.1 Backend
- NestJS (TypeScript)
- Prisma ORM
- PostgreSQL

### 5.2 Frontend
- Next.js 또는 React
- TailwindCSS

### 5.3 블록체인
- Solana (Devnet → Mainnet)
- @solana/web3.js

### 5.4 기타
- PDF 생성: pdf-lib 또는 puppeteer
- 암호화: AES-256 (개인정보)

## 6. MVP 범위

### 6.1 포함
- 계약서 CRUD
- 고용주/근로자 서명
- PDF 생성 및 다운로드
- Solana Devnet 해시 기록
- 기본 검증 페이지

### 6.2 제외 (Phase 2)
- 본인인증 (PASS/카카오)
- Solana Mainnet
- 알림 서비스 (SMS/카카오톡)
- 계약 만료 알림
- 다국어 지원

## 7. 데이터 요구사항

### 7.1 필수 입력 항목
- 고용주: 상호명, 대표자명, 사업장 주소, 연락처
- 근로자: 이름, 생년월일, 연락처, 주소
- 계약: 계약 유형, 시작일, 근무 요일, 근무 시간, 시급, 급여일

### 7.2 선택 입력 항목
- 종료일 (무기한 계약 가능)
- 특약 사항

## 8. 보안 요구사항

- 개인정보 암호화 저장 (AES-256)
- HTTPS 필수
- 서명 데이터 무결성 검증
- 접근 권한 관리 (본인 계약서만 조회 가능)

## 9. 성공 지표

- 계약서 작성 완료율
- 서명 완료까지 소요 시간
- 블록체인 검증 요청 수
- 사용자 재방문율
